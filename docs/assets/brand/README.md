# Aventine brand assets

The primary Aventine identity is a modern evaluation seal built around a high-ground beacon. The
hill represents perspective and a common benchmark; the small diamond at its summit represents the
signal that makes results legible and comparable.

| --- | --- | --- |

SVG is the source of truth. Use PNG only where vector images are unsupported.

## Palette

- Aventine gold: `#C8A45B`
- Pale gold: `#D7B96F`
- Warm ivory: `#F7F1E6`
- Midnight: `#161C24`
- Charcoal: `#2A2E35`

## Usage

- Prefer the complete seal for repository, publication, and profile imagery.
- Use the standalone mark only for small icons or places where the circular typography will not be readable.
- Use the compact avatar for square profile images and favicon-sized placements.
- Keep clear space around the logo equal to at least one beacon-diamond height.
- Do not stretch, rotate, recolor individual elements, or add effects and gradients.
- The SVG wordmark uses a Baskerville/Georgia serif stack and a restrained sans-serif stack. The
  supplied PNG exports preserve the approved rendering when exact cross-platform consistency matters.
